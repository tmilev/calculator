This file shows how to bundle the equation editor with webpack. It uses the files: 
    equation_editor.js
    app.js
The entry point file is app.js, specified in file webpack.config.js, used to generate the app_minified.js script, which is then included in the present html file. The app.js file holds all code that initializes the page.
