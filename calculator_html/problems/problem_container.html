The problem content must be in a separate page: its source contains the problem answer which must be stripped before showing to the student. Include the problem by specifying its file name in a tag named "problem". Then call the javascript: window.calculator.problemPage.loadProblemIntoElement(element), where the input is the HTMLEelement (or its id) where the problem is to be displayed. Note: the calculator javascript library installs itself in the window global variable window.calculator.
