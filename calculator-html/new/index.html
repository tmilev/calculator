Project mission

The mission of the calculator project is, to the extent possible,
to develop a universally accessible, free and open source computer algebra system covering all computational aspects of mathematical research and math education, using an intuitive and user friendly interface that gives full programmatic control over the system to its users.

About the software

The source code of the calculator can be found here.
Calculator source code
The code is to be compiled on a Linux machine.

Most of the functionality of the system should work out of the box (if not, it's a bug). However, for obvious reasons, parts of the calculator will not work immediately - for example, the officially signed security certificates of calculator-algebra.org are kept secret. If you need help setting up the calculator on your machine, write me an email (todor.milev at google's email).

Software license

The software is licensed to you under the Apache License, Version 2.0. This a permissive free software license which, roughly speaking, states that you are free to use the calculator, install it on your machine, use it to run your website, modify its source code to suit your needs and redistribute the resulting work.

The calculator is fully functional without any homework problem files, and provides facilities for creating such files (I create my own homework files through the calculator interface). Should you install the calculator on your own machine and create your own homework problems, you own the copyright to the resulting files; those are not subject to the Apache License.

The homework problems located at www.calculator-algebra.org are private and are not subject to the Apache License.
